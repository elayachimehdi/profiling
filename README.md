# profiling


Le fichier data_preparation est le fichier executant la preparation des données. Ce script utilise les modules (fichiers) fonctions et changements. Il se connecte à la base de données et l'importe. Ensuite, après avoir converti la base de données en DataFrame pandas, ce script commence à effectuer ses traitements.

Le fichier fonctions regroupe certaines fonctions afin de pouvoir factoriser notre code

Le fichier changements contient des listes des changements à réliser dans la base de données.
